# Calatrava-MacDown-Html-Transformation

将`MacDown`输出的 HTML 转为`Calatrava`可用的 HTML 页面。


# 使用方法

```bash
$ path/for/Calatrava-MacDown-Html-Transformation [inputpath] [outputpath]
```
